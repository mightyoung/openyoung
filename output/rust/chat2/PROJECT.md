# Rust ChatBot 项目说明

## 项目概述
这是一个用 Rust 编写的简单命令行聊天机器人，展示了 Rust 的基本语法、结构体、向量、模式匹配和外部 crate 的使用。

## 核心功能

### 1. 对话系统
- 关键词匹配响应
- 默认回复机制
- 对话历史记录

### 2. 实用功能
- 时间查询（使用 chrono crate）
- 日期查询
- 随机笑话（使用 rand crate）

### 3. 历史管理
- 对话记录存储
- 历史查看功能
- 历史清理功能

## 代码结构

### main.rs 主要组件

1. **ChatBot 结构体**
   - 存储机器人名称
   - 关键词-响应映射
   - 笑话库
   - 对话历史

2. **主要方法**
   - `new()` - 初始化机器人
   - `respond()` - 处理用户输入并生成响应
   - `add_to_history()` - 记录对话

3. **用户界面**
   - 美观的欢迎界面
   - 交互式命令行
   - 清晰的输出格式

## 学习要点

这个项目展示了以下 Rust 概念：

1. **结构体和实现** - 定义 ChatBot 数据结构和方法
2. **向量 (Vec)** - 存储响应、笑话和历史记录
3. **字符串处理** - 大小写转换、包含检查
4. **模式匹配** - 处理特殊命令
5. **外部 crate** - 使用 chrono 和 rand
6. **错误处理** - 使用 expect 处理输入错误
7. **循环和控制流** - 主对话循环

## 运行说明

```bash
# 构建项目
cargo build

# 运行项目
cargo run

# 或使用脚本
./run.sh
```

## 依赖管理

项目使用 Cargo 管理依赖：
- `chrono = "0.4"` - 日期时间处理
- `rand = "0.8"` - 随机数生成
