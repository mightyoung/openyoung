### `SkillLoader`

Skill 加载器 - 渐进式披露 + 统一检索

**Methods:**
- `get_metadata`
- `list_all_metadata`
- `check_requirements`
- `get_always_skills`
- `get_loaded_skills`

## Functions

### `get_metadata()`

获取 Skill 元数据（不加载内容）

### `list_all_metadata()`

列出所有 Skill 元数据

### `check_requirements()`

检查 Skill 依赖是否满足

Returns:
    (是否满足, 缺失的依赖列表)

### `get_always_skills()`

获取总是加载的 Skills

### `get_loaded_skills()`

获取所有已加载的 Skills
