### `AgentQualityReport`

### `AgentEvaluator`

Agent 质量评估器

评估维度：
1. COMPLETENESS - 文件完整性
2. VALIDITY - 配置有效性
3. DEPENDENCIES - 依赖完整性
4. DOCUMENTATION - 文档质量
5. SECURITY - 安全检查
